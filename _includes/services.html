Vision

Combining ML and physics could reveal the best of both worlds.

Physics

Physical models have been used to describe the world around us. If we have sufficient information about a process, a well-made physical model can accurately predict future events. These models have been used for vastly differenct processes ranging from predicting the trajectory of a rocket to predicting the behavior of nano-scale objects. However such models are difficult to define when we have no direct understanding of the underlying scenario.

Machine Learning

Techological advancements have made possible the collection of vast amounts of data and a dramatic increase in computational power has created the possibility of analyzing collected data in reasonable time using modern statistical and machine learning (ML) approaches. Even though such data-driven models have outperformed physics based models in many areas, they are data hungry and generalization remains to be a challenge.

Physics Based Learning Models

What if we could combine the computational data-driven method of machine learning with the scientifically driven method of physical models? Machine learning algorithms learn by searching for an optimal solution in the optimization space defined by our neural network architecture, inputs and outputs. On their own, machine learning algorithms only use data as a guide to search for an optimial solution. We could however develop models, known as Physics Based Learning Models which use the knowledge of the world, that is, physics together with data to look for an optimal solution.
